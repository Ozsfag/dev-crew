## Контекст

CLAUDE.md требует 100% покрытия кода. Сервисы `audit/` реализованы полностью,
но не имеют unit-тестов. `AuditPostAgentHook` уже покрыт (✅). Непокрыты

## Проблема

Непокрытые классы:
- `audit/app/service/command/AuditCommandService.java` — 1 публичный метод `record()`
- `audit/app/service/query/AuditQueryService.java` — 2 публичных метода: `findByTimestampBetween()`, `findByProjectId()`

Критическая ветка без теста: `if (projectId != null)` в контроллере — вызывает разные методы сервиса.

### AuditControllerTest

Standalone MockMvc без `@SpringBootTest`. Ключевой момент — покрыть оба пути ветвления:

```java
// audit/adapter/in/web/AuditControllerTest.java
@ExtendWith(MockitoExtension.class)
class AuditControllerTest {

  @Mock private AuditQueryService queryService;
  private MockMvc mockMvc;

  private static final Instant FROM = Instant.parse("2026-01-01T00:00:00Z");
  private static final Instant TO   = Instant.parse("2026-01-01T23:59:59Z");

  @BeforeEach
  void setUp() {
    var controller = new AuditController(
        queryService,
        Mappers.getMapper(AuditWebMapper.class));
    mockMvc = MockMvcBuilders.standaloneSetup(controller)
        .setControllerAdvice(new GlobalExceptionHandler())
        .build();
  }

  @Test
  void GET_audit_returns_events_without_projectId() throws Exception {
    when(queryService.findByTimestampBetween(FROM, TO)).thenReturn(List.of(auditEvent()));

    mockMvc.perform(get("/api/audit")
            .param("from", FROM.toString())
            .param("to", TO.toString()))
        .andExpect(status().isOk())
        .andExpect(jsonPath("$").isArray());

    verify(queryService).findByTimestampBetween(FROM, TO);
    verify(queryService, never()).findByProjectId(any(), any(), any());
  }

  @Test
  void GET_audit_returns_events_filtered_by_projectId() throws Exception {
    var projectId = UUID.randomUUID();
    when(queryService.findByProjectId(projectId, FROM, TO)).thenReturn(List.of(auditEvent()));

    mockMvc.perform(get("/api/audit")
            .param("from", FROM.toString())
            .param("to", TO.toString())
            .param("projectId", projectId.toString()))
        .andExpect(status().isOk());

    verify(queryService).findByProjectId(projectId, FROM, TO);
    verify(queryService, never()).findByTimestampBetween(any(), any());
  }

  private AuditEventModel auditEvent() {
    return new AuditEventModel(
        UUID.randomUUID(), UUID.randomUUID(),
        "user@example.com", "TASK_CREATED",
        UUID.randomUUID(), "details",
        Instant.parse("2026-01-01T10:00:00Z"));
  }
}
```
